# ⚙️ CodeShifter

CodeShifter is a Python-based refactoring tool with a **Flask backend** and a future **frontend interface**.  
It helps developers reorganize project structures by moving files, updating imports, cleaning empty directories, and ensuring `__init__.py` files exist.  

---

## 🚀 Features

- 📂 Move files into new folder structures  
- 🔄 Update Python imports automatically  
- 🧹 Clean up empty directories or directories with only `__init__.py`  
- 📝 Automatically add `__init__.py` files in new folders  
- 🧪 Dry-run mode for safe previews
